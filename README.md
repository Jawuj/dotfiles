- **WM / Compositor:** Hyprland (Wayland)
- **GPU:** NVIDIA (Oficial Drivers)
- **Login Manager:** Ly
- **Bar:** Waybar
- **Launcher:** Rofi
- **Lock / Logout:** Hyprlock · Wlogout
